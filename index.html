Projetos

Projeto 1: Aplicativo de Gerenciamento de Finanças Pessoais

Um estudante de engenharia de software desenvolveu um aplicativo móvel de gerenciamento de finanças pessoais. O aplicativo permite aos usuários controlar suas despesas, criar orçamentos e acompanhar seu progresso financeiro. Foram utilizadas tecnologias como React Native e Node.js para desenvolver a interface do usuário e a API de backend. O projeto proporcionou ao estudante a aplicação de conhecimentos full-stack e a compreensão da importância da gestão financeira pessoal.

Projeto 2: Sistema de Reservas de Hotéis Online

Em um projeto de equipe, um estudante de engenharia de software participou do desenvolvimento de um sistema de reservas de hotéis online. Sua contribuição incluiu a implementação da interface do usuário usando HTML, CSS e JavaScript, além da integração com a API de backend. O sistema permitia pesquisar hotéis, fazer reservas e obter informações sobre quartos e serviços. O estudante também trabalhou na implementação de um sistema de autenticação seguro. O projeto proporcionou experiência em trabalho em equipe e aprimorou habilidades de front-end e integração com APIs.
